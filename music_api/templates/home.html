О проекте

RubySound.fm — веб-приложение на Django и REST Framework для поиска музыки и артистов, а также для прослушивания превью треков. Проект объединяет данные из Last.fm, Deezer и iTunes, предоставляет REST API и интерактивный фронтенд с возможностью прослушивания превью и анимациями.
